# 宠物健康计划 Web - 产品需求文档 (PRD)

> **当前状态**: MVP 前端演示版 (Mock Mode) 已完成
> **最后更新**: 2026-03-10

## 1. 产品概述
打造一款集宠物信息管理、AI 智能健康规划、硬件联动（MCP）于一体的宠物健康管理 Web 应用。通过科学的数据分析和自动化设备，实现宠物的精细化饲养。

**当前阶段重点**：验证前端交互流程与UI体验，使用本地模拟数据（Mock Data）替代后端服务。

## 2. 用户角色
- **宠物主**：添加宠物、制定计划、查看报表、控制设备。
- **管理员**（可选）：后台维护数据。

## 3. 核心功能模块

### 3.1 宠物档案管理 (Pet Profile) ✅ [已实现]
- **基础信息**：
  - 头像上传（支持本地图片预览与 Base64 存储）✅
  - 昵称、品种（猫/狗）、性别、出生日期、体重。
- **健康信息**：绝育状态、既往病史（文本记录）。
- **操作**：
  - 新增宠物：支持完整表单录入。
  - 编辑宠物：支持回显已有信息并修改。
  - 删除宠物：二次确认后删除。
  - **多宠物切换**：在仪表盘和侧边栏支持快速切换当前管理的宠物。

### 3.2 AI 健康计划 (AI Health Plan) 🚧 [开发中]
- **目标设定**：
  - 减重模式（当前体重 -> 目标体重）
  - 增重模式
  - 保持健康模式
- **AI 生成**：调用大模型生成每日饮食计划（时间、分量）、饮水目标、运动建议。
- **计划确认**：用户确认 AI 生成的建议后，计划生效，生成每日任务。
- **动态调整**：根据周报表（体重变化、进食频率）自动建议调整计划。

### 3.3 每日追踪与仪表盘 (Dashboard) ✅ [已实现]
- **今日任务队列**：
  - **队列显示**：仅展示前 3 项待办任务，保持界面简洁。
  - **任务交互**：点击完成任务后，任务自动移动至队列末尾并变灰，新任务自动补位。
  - **状态统计**：实时显示“待办 X 项”与“已办 Y 项”。
- **打卡记录**：
  - 快速状态记录：一键记录“开心”、“玩耍”、“睡觉”等状态。
- **健康报表**：
  - **体重趋势图**：基于 Recharts 实现的最近 7 天体重变化曲线。
- **环境信息**：模拟显示当前天气与温度（UI 展示）。

### 3.4 硬件控制中心 (Device Control) 🚧 [规划中]
- **设备绑定**：绑定智能投食器、喂水器（模拟）。
- **远程控制**：
  - "立即投食"（克数）
  - "出水"（毫升）
- **状态监控**：设备在线状态、剩余粮量/水量。

## 4. 交互流程
1. **初始化**：用户登录（模拟） -> 创建/查看宠物档案。
2. **日常使用**：
   - 进入仪表盘，查看今日首要任务。
   - 完成喂食/互动任务，点击打卡。
   - 记录宠物最新体重，查看趋势变化。
3. **档案维护**：
   - 上传宠物萌照作为头像。
   - 更新宠物体重或健康状况。

## 5. 界面规范
- **风格**：温馨（暖橙色为主）、简洁、现代卡片式设计。
- **布局**：
  - **移动端优先**：底部导航栏（Dashboard, Plans, Devices, Profile）。
  - **响应式**：适配桌面端宽屏显示。
- **交互**：
  - 使用平滑动画（Framer Motion / CSS Animation）增强任务完成和页面切换体验。
  - 即时反馈：表单提交、任务完成均有视觉反馈。
